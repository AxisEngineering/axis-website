AXIS WEBSITE - LIVE RECHECK AFTER CLAUDE UPDATE
Date checked: 2026-07-02
Site checked: https://www.axisengineering.co.uk/

SUMMARY VERDICT
The live homepage has been substantially improved. The main technical wording
changes have been made and the most risky overclaims are no longer present.

However, the wider SEO, privacy and accessibility work from the implementation
brief has not been completed yet. At present this looks like a good homepage
copy update, not a full site/SEO implementation.


WHAT IS NOW FIXED / IMPROVED

1. Technical wording is much safer
The live homepage now includes the corrected wording:
- "No gears. No friction drive. No drive-contact wear."
- "Project duty 0 - 100+ m/s"
- "Typical 2 - 40 mm mechanical"
- "Field speed, current & slip"

The slip explanation is also improved. It now refers to normal motoring and
mentions thermal, gap and saturation limits.

The DSLIM section now says the design can approach twice the active-face thrust
under comparable current, gap and thermal limits, and that opposing normal
attractions largely cancel. This is much better than the earlier absolute claim.

The permanent-magnet brake section now avoids the old "faster means harder
braking" overclaim and includes the important note that a dynamic brake does not
provide held standstill restraint.

2. Old risky phrases are gone
I checked the live HTML for the following phrases and found none:
- "vehicle never carries"
- "the faster it goes the harder it brakes"
- "still brakes on power loss"
- "plate always runs slower"
- "No contact wear"
- "fling coaster"
- "only thing left to push"
- "twice the thrust of a single-sided"

3. Redirects are working
The following routes resolve to the main live site:
- https://axisengineering.co.uk/
- https://www.axisengineering.co.uk/
- http://www.axisengineering.co.uk/
- https://axisengineering.github.io/axis-website/

4. Mobile and desktop rendering checks passed
I checked the live site at:
- 1440 px desktop
- 768 px tablet
- 390 px mobile
- 360 px mobile

Results:
- HTTP 200 response
- No JavaScript console errors
- No horizontal overflow
- Mobile menu opens
- Menu button is 44 x 44 px
- Hero text is readable on narrow mobile screens


WHAT IS STILL MISSING

1. Planned SEO pages are not live
The following planned URLs still return 404:
- /linear-induction-motors/
- /linear-induction-motor-design/
- /permanent-magnet-brakes/
- /eddy-current-brakes/
- /double-sided-lim-dslim/
- /applications/people-movers/
- /applications/launch-systems/
- /applications/maglev-propulsion/
- /technical-resources/
- /privacy/

This means the site is still mostly a single-page website, so the bigger SEO
opportunity has not been delivered yet.

2. Sitemap has not been updated
The live sitemap still lists only the homepage:
- https://www.axisengineering.co.uk/

It does not include the planned SEO or privacy pages.

3. Privacy/compliance work is still missing
The privacy page is not live.

The contact form still does not include a short privacy notice under the submit
button, and there is no visible Privacy Notice link near the form.

This is still important because the form collects personal data and uses
Web3Forms.

4. Contact form accessibility is still incomplete
The form fields still rely on placeholders and aria-labels. There are no visible
labels for:
- Name
- Company
- Email address
- Application type
- Duty/project description

This was specifically requested in the implementation brief and has not been
done.

5. Skip link is only partially fixed
A skip link exists, which is good, but it points to #top and the page does not
contain a proper <main> element.

Better implementation would be:
- Add <main id="main">
- Point the skip link to #main

6. Structured data remains basic
The site has Organization JSON-LD, but it does not include a contactPoint.

Recommended next improvement:
- Add contactPoint with sales email and enquiry purpose
- Consider adding LocalBusiness/EngineeringService style details if appropriate

7. Metadata has not really been improved
The title and meta description are still broadly the same:
- Title: Axis Engineering - Linear Induction Motors & Magnetic Brakes
- Description: Axis Engineering designs and manufactures linear induction
  motors (LIMs) and permanent-magnet brakes for transit, launch and industrial
  systems.

This is acceptable, but it does not yet capture the fuller SEO positioning
around UK design/manufacture, eddy-current brakes, maglev, people movers,
launch systems and technical design services.


CONCLUSION
Claude appears to have done the homepage wording pass well. The website is now
more technically credible and reads better.

The remaining work is not cosmetic. The missing SEO pages, privacy page,
sitemap update, visible labels and stronger metadata are still the main things
holding the site back from being a fully polished business website.

Recommended next instruction:
"Keep the current homepage. Now complete the outstanding implementation brief:
create the planned SEO pages and privacy page, update sitemap.xml, add visible
form labels and privacy text, improve metadata, and add richer structured data."
